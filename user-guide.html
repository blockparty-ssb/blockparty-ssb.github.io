Why do I have to make an account on Digital Ocean?

Digital Ocean is a hosting company and will host your pub. This pub is like an always-online group member whose task is to replicate all of your messages - like a post office which delivers the letters to all the receivers.


Without this pub, exchanging messages would take a very long time and only happen if you and your friends are online at the same time.


Each community space has its own pub. If you want to start another community space you need to setup another pub.

Does it cost any money?

Using blockparty is for free, as it is non-commercial and open source software. Using Digital Ocean as host for your pub does cost depending on how much data storage you use. For 1 GB of storage space you will pay not much more than for a "Herrengedeck" in your favourite "Eckkneipe" - around $5/month. (For detailed information, please check the pricing.) You will need to choose a payment method (credit card or paypal) and add your card data or your paypal account respectively.

How to create and access my DigitalOcean token?

Personal access tokens function like a combined name and password for API authentication. You will need the token at the time you create a new blockparty community. When logged in to your Digital Ocean account, you can generate a new token with this link.
https://cloud.digitalocean.com/account/api/tokens/
Name it as you like. After creating the token, copy and save it on your computer, as it will not be shown again. Keep your tokens secret. They function like passwords. Please do not share it publicly. Read more about API tokens here. If you loose your API key, you can always create a new one.
